## 介紹

DND5E城主工具包，實現了以下功能：
- **寶石、DMG魔法物品隨機表**。作者：Dora、Eternal Rider，該功能依賴Better Roll Tables
- **隨機名稱產生器**。作者：Eternal Rider，該功能依賴Better Roll Tables
- **一鍵產生商人及物品**。作者：Eternal Rider，該功能依賴Better Roll Tables 以及Loot Sheet NPC 5E
